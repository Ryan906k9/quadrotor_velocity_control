# 深度学习算法控制四轴飞行器速度

使用深度学习算法，通过控制 4 个马达的电压，达到对四轴飞行器速度的自主控制

# 算法创新

传统的方法会使用 4 个维度的输出控制 4 个马达

但是这种方法使用深度学习算法达到收敛至少要 80-100w steps 左右

这里我们使用 5 个维度的输出，利用基础电压 + 修正电压的方式控制 4 个马达

这个修改可以让深度学习算法在 3w-5w steps 左右就开始快速收敛

把训练速度提升 20 倍！
